# Smart Fraud Detection System for Banking Transactions

## 📌 Project Description
This is a Flask-based web application that detects fraudulent banking transactions using risk scoring and visual analytics.

## 🚀 Features
- Transaction fraud detection
- Risk score calculation
- Transaction history stored in CSV/SQLite
- Dashboard with charts (bar, pie)
- Flask web interface

## 🛠️ Technologies Used
- Python
- Flask
- HTML, CSS
- Matplotlib
- SQLite / CSV
